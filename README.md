# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data using SQL and Tableau to identify patterns that contribute to customer loss. The goal was to uncover high-risk customer segments and provide business recommendations to improve customer retention.

---

## Business Problem
Customer churn directly impacts revenue and long-term business growth. This analysis helps identify:
- Which customers are most likely to churn
- Key factors driving churn
- Opportunities to improve customer retention strategies

---

## Tools Used
- MySQL
- Tableau
- Excel
- GitHub

---

## Dataset
- 7,000+ customer records
- Includes:
  - Contract type
  - Monthly charges
  - Payment method
  - Tenure
  - Internet service
  - Churn status

---

## Key Questions Answered
1. Which contract types experience the highest churn?
2. Does monthly cost impact churn rate?
3. Which payment methods correlate with churn?
4. Are long-term customers less likely to churn?
5. Which customer segments are highest risk?

---

## SQL Skills Demonstrated
- Joins
- Aggregations
- GROUP BY
- CASE Statements
- Filtering
- KPI calculations
- Churn rate analysis

---

## Key Insights
- Month-to-month contracts showed the highest churn rates.
- Customers with higher monthly charges were more likely to churn.
- Electronic check payment methods correlated with increased churn.
- Long-term customers demonstrated significantly stronger retention.

---

## Business Recommendations
- Incentivize customers to move to long-term contracts
- Improve retention offers for high-risk customers
- Create targeted campaigns for short-tenure customers
- Reduce friction in payment processing experience
